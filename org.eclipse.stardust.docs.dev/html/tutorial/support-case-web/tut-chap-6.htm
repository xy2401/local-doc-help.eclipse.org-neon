Creating Structured Data

Once composite types are created, you can create structured data using the same. Let's create a structured data named Support Case Data.

To create Support Case Data:

  1. In the My Process Models panel, expand the Support-Case model node
  2. Right-click the Data node and select the Create Structured Data option. The Structured Data 1 is created.
  3. In the General Properties section, specify name as Support Case Data
  4. In the Data Type panel, select the Structured Data option from the drop-down list
  5. Select Customer from the Structured Type drop-down list
