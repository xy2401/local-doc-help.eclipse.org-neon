5.1.4.8.2. Check Text in Tab Components

Check Text of Tab by Index

  • Use this action to check the text of a tab in a tabbed pane.

  • You enter the index of the tab you want to check and the text you expect.

  • Parameters:

    • Index

    • Title

    • Operator
