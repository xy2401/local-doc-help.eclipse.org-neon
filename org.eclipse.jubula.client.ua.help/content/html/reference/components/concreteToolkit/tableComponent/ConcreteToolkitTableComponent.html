3.2.9. Concrete toolkit table component

A table component is a component in which data is displayed and edited.

The format of a table component is a two-dimensional layout of cells which are organized into columns and rows.

Please keep in mind that it is not possible to object map a single cell within the table component but to map the table component as a whole only. In order to interact with certain cells within the table component you have to use the according actions.
