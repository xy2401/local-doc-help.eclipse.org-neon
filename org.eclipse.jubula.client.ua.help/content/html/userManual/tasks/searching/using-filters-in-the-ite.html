3.29.9. Using filters in the ITE

3.29.9.1. Text filters

The Test Case Browser, the Test Suite Browser, the Component Names Browser and the Object Mapping Editor (in the tree view) all contain a filter area at the top. You can enter a filter text into this area to show matches to the text in the browser or editor. Use star (*) as a wildcard.

In the Test Case Browser, the Component Names Browser and the Test Suite Browser, the names of the items in the browser are considered for the filter.

In the Object Mapping Editor, the names and types of the components are considered for the filter.
