On Using Bookmarks

The Process Model Bookmarks view gives you the opportunity to cluster the components of your model in arbitrarily organized groups of tools. The upper tree view organizes the elements of the model by categories (e.g. by participants, processes or data). For example, using bookmarks you can organize the most frequently used components under a bookmark with an arbitrary name, like My Tools.

Creating a Bookmark

To create a bookmark in the Process Model Bookmarks view, click the model and select Add View from the pop-up menu.


Figure: Creating a New Bookmark

To rename the newly created bookmark:

  1. Right-click the model and choose Rename from the context menu.
  2. Enter My Tools in the name field.
  3. Drag an activity, a participant, a data or an application symbol from the Outline view onto the My Tools bookmark symbol and drop it there.


Figure: Adding Elements to an Individual Bookmark

The content of the bookmark can be filled by dragging the selected elements from the Outline view area dropping them on the bookmarks view. Likewise, by drag-and-drop, you can - fast and easily - get your model elements from the bookmark into diagrams you are building.
