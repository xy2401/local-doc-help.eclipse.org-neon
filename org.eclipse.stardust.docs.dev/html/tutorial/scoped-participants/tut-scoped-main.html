Scoped Participants Tutorials

These tutorials demonstrate the usage of departments as runtime instances of organizations.

The examples guides through the following use cases:

  • The Scoped Participant Example - is a simple use case created to explain how to develop a multi-scoped organizational structure and work with multi-scoped participants.
  • Use Case - Delegating to a User in Another Target Department - a simple use case example on how to delegate a user to a user in another target department.

Example Sources

The complete example models are provided by Stardust. In case you like to use or view the completed model files, click the following link to download the ZIP file containing the sources in folder scopedParticipants:

  • all-tutorials.zip

Folder scoped-participants of the ZIP file contains the example models, which are used in the according tutorials.
