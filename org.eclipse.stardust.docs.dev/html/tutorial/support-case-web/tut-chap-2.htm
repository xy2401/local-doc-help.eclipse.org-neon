Creating a Process

To define the workflow, you need to create a process definition. Wherein you can add and connect model elements.

To create a process:

  1. In the My Process Models panel, right-click the model name and select the Create Process option.


    Figure: Create Process

  2. The Process 1 gets created under the model node.


    Figure: Process 1

  3. Rename the process as Support Case Management

Process Diagram Canvas

Each process has a Default Pool and a Default Lane. You can add more than one swim lanes to the default pool and assign participants for each swim lane. Each process has its property panel wherein you can define properties for the selected process.
