4.2.2. Supported Swing AUTs

AUTs written with the Swing GUI toolkit are supported according to the following points:

  • The AUT is written using Java 1.5 or higher.

  • It uses a Java SE (Standard Edition) JRE.

    AUTs based on the NetBeans framework, and NetBeans itself, are currently not supported.
