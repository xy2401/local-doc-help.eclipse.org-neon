Adding Process Data Paths

Let's add process data paths to our Support Case Management process, which we can use to create correspondence in the Stardust portal later.

Creating a Data Path - CustomerName

To create a data path:

  1. Open properties of the Support Case Management process definition.
  2. Click the Data Path icon on the left side of the properties panel of a process.
  3. Click the Add icon below the Data Path list. A new data path with a default name is created.
  4. Specify CustomerName in the Name field.
  5. Ensure that Direction is In and Type is Standard.
  6. Select Support Case Data from the Data drop-down list.
  7. Select Name from the Path drop-down list.


    Figure: Data Path - CustomerName

Creating a Data Path - ProductName

To create a data path:

  1. Click the Add icon.
  2. Specify ProductName in the Name field.
  3. Ensure that Direction is In and Type is Standard.
  4. Select Support Case Data from the Data drop-down list.
  5. Select Product from the Path drop-down list.
  6. Add dot in front of the Product path. The populated list displays fields in the Product data.


    Figure: Product - Populated List

  7. Select Name from the populated list.


    Figure: ProductName
