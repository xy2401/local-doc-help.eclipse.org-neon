5.3.2.2.1. Check Editability in Tree Tables

Check Editability (Specify Cell)

  • Use this action to check if a cell is editable.

  • You enter the cell you want to check and whether you expect it to be editable or not.

  • Parameters:

    • Editable

    • Row

    • Row Operator

    • Column

    • Column Operator

Check Editability of Cell (Mouse Position)

  • Use this action to check whether a cell at the current mouse position is editable or not.

  • Parameters:

    • Editable

Check Editability of Selected Cell

  • Use this action to check whether the currently selected cell is editable or not.

  • Parameters:

    • Editable
