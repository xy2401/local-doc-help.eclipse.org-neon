4.3.4. Keyboard Layouts

For RCP AUTs, a keyboard layout must be entered in the AUT configuration (Section 3.8.4.4, “Advanced AUT configuration”). German (DE) and English (US) are provided as standard keyboard layouts.

If you require a different keyboard layout, you must create a mapping file for the new language and place it in the server/resources directory. You also have to put this mapping file into the RCP Remote Control plug-in of your AUT:

<RCPAUT>/plugins/org.eclipse.jubula.rc.rcp_<Version>/resources
