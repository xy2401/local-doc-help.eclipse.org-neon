Modeling Guide

The Modeling Guide explains the basic concepts of the Stardust Process Engine and provides an in-depth view of how to model your business processes with Stardust.

Audience

The Modeling Guide is intended to be read by people who do process modeling with Stardust or use Stardust as a programmer.

Contents

The book is organized into the following chapters:

  1. Chapter The Process Workbench serves as a user guide to model processes with the Stardust Process Workbench.
  2. Data Integration describes the possibilities to integrate specific data types.
  3. System Integration describes the possibilities to integrate your existing IT infrastructure as a process modeler.
  4. Chapter The Model File Format describes the model file formats and their handling.
