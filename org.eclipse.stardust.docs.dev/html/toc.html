Stardust Documentation

The Stardust documentation suite for Release 4.1.0 contains a set of user guides, a JavaDoc collection of class descriptions as well as the release notes.

Release Notes

The Release Notes provide the release notes for the actual release and links to earlier release notes.

Upgrading to Later Versions

Please refer to chapter Upgrading to Later Versions for detailed information on required changes on existing models when upgrading to a later version of Stardust.

Installation

The Installation Guide provides general information about the server installation and database configurations.

Getting Started

Chapter How to read the Stardust Documentation gives a brief introduction to the Stardust Documentation.

The Tutorials guide you through the Stardust functionality with example use cases.

The chapter Self-Learning Training Videos provides an overview of links to a series of self-learning videos to learn how to use Stardust.

Concepts

The Concepts section helps you understand the key concepts of Stardust.

Administrator Handbook

The Administrator Handbook provides an overview over installation and operations of Stardust.

Model & Go! Handbook

The Model & Go! Handbook provides details on the functionality of the Model & Go! perspective, which supports building and managing simple Process Definitions in the Portal.

Business Process Modeling Handbook

The Business Process Modeling Handbook contains a set of user guide for business analysts for modeling and installation.

Developer Handbook

The Developer Handbook covers the following topics:

  • Modeling Guide
    Introduction of modeling basics and integration issues.
  • Operation Guide
    Provides descriptions of administration facilities.
  • Programming Guide
    Describes the basic concepts of the Stardust API and its relationship to the general process modeling and runtime concepts.
  • Deployment Guide
    The Deployment Guide provides information on how to deploy and configure a Stardust runtime environment.
  • Document Service Integration Guide
    The Document Service Integration provides the usage of the functionality for content repositories of document management systems.
  • Simulation Feature
    The Simulation Feature allows a process model to be tested for performance characteristics without actually deploying the process.
  • Spring Integration Guide
    The Spring Integration Guide helps you to get started with using Stardust in a Spring environment.
  • Rapid Application Development
    The Rapid Application Development offers the possibility to get started quickly in application development without requiring a client-dependent development environment.
  • Stardust Schema
    Provides an overview on the Stardust schema.

End User Handbook

The End User Handbook contains a set of user guides for end users, like the description of the Stardust Web Portal.

Using Stardust for System Integration

The handbook Using Stardust for System Integration provides a user guide for system integrating with Stardust. It describes integration tasks, configuration tasks, development as well as deployment, monitoring and diagnostics.

Examples

The Examples section provides examples for Web services API, authentication and synchronization (security).

Reference

The Stardust Reference part contains the Stardust API description and the Stardust extension points references.

Troubleshooting

Please refer to the Troubleshooting chapter for known or restricted issues and provided workarounds.

Glossary

The Glossary provides an alphabetical list of terms and their definitions.
