Administrator Handbook

Audience

This handbook contains a set of user guides for administrators.

Contents

This book is organized into the following topics:

  • Upgrading to Later Versions: describes changes required for upgrading from earlier Stardust versions.
  • Troubleshooting: provides a list of known issues and known restrictions for the usage of Stardust.
  • Installation Guide: provides general information about the server installation and database configurations.
  • Operation Guide: describes the deployment to various application servers and administration facilities.

Related Documents

Chapter Administration in the Stardust Portal documentation provides details on administrative actions in a Web-based environment.
