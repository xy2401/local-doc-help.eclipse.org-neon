4.85. Type

Parameter Name Data Type Possible Values Default
TYPE String

  • ACCEPT

  • DISMISS

 none
Table 4.86. TYPE

  • Choose the way you want to handle Alert/Prompt/Confirm dialogs. You have the options to accept or dismiss them.

  • E.g.: If a Prompt dialog has the options to click OK and Cancel ACCEPT would click OK and DISMISS would click Cancel.
