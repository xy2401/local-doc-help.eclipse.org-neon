Tuning

Tuning Tasks

The Stardust Process Engine is designed to satisfy highest performance and concurrency requirements.

Most of the operations performed with Stardust Process Engine do not have a significant impact on the performance and the concurrent throughput of the software systems embedding the Stardust Process Engine. However, some operations can be expensive in terms of execution time or concurrent access, e.g. extensive use of data values in worklist queries may have visible impact on the query execution time. For those operations Stardust provides powerful tuning facilities as described in this chapter.
