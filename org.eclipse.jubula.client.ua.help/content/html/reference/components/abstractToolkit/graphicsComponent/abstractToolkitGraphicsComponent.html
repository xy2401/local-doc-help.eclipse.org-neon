3.1.2. Abstract toolkit graphics component

This is an abstract component.

It represents the basic properties of all of the objects in an application, for example:

  • if they are enabled (manipulable)

  • if they are selected (have focus)

  • other properties (e.g. size, shape, color)

Available Actions

  • Check Enablement (General Components)

  • Check Existence (General Components)

  • Check Focus

  • Check Property

  • Check Selection of Context Menu Entry (General Components)

  • Click

  • Drag and Drop (General Components)

  • Invoke Method (General Components)

  • Select in Context Menus

  • Show text

  • Store Property/Value (General Components)

  • Wait for Component (General)
