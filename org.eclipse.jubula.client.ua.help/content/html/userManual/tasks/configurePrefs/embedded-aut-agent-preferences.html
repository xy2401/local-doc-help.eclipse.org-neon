3.28.4. Embedded AUT Agent preferences

You can find the preference page for the embedded AUT Agent under:

Test-AUT Agent-Embedded AUT Agent

in the preferences dialog.

On this page, you can enter the port number that the embedded AUT Agent (Section 3.2.3, “Connecting to the AUT Agent”) should use when you connect to it (Section 3.2.3, “Connecting to the AUT Agent”). The default is 60001.
