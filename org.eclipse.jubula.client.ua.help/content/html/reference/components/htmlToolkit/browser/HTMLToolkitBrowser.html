3.4.3. HTML toolkit browser

The Web Browser component is the container for all other components in a HTML AUT.

It represents the AUT as a whole.

Available Actions

  • Go Back

  • Open URL

  • Reload page

  • Select Window by Title

  • Select Window with Identifier
