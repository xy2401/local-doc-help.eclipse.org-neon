Log Entries

Stardust creates log entries for all relevant events in a runtime environment, such as:

  • user logins and log outs, including failed logins,
  • daemon starts and terminations,
  • exceptions in user applications coordinated by Stardust and
  • exceptions thrown in the runtime environment.

A log entry contains information about its:

  • creation time,
  • scope (activity instance, process instance or global) and
  • context information (error messages etc.).
