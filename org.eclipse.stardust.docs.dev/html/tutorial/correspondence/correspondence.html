Creating Correspondence

In this tutorial we prepare correspondence by using different correspondence templates. To use correspondence, Stardust provides a predefined structured data type CORRESPONDENCE.xsd containing required correspondence data types. We will create a model with a correspondence process, which will use this predefined data types and a UI Mashup for correspondence.

You can download the tutorial zip file containing the predefined Correspondence data model and the correspondence example model from here:

all-tutorials.zip

You find the example models in folder correspondence.

For details on creating correspondence, please refer to chapter Creating Correspondence in the End User Handbook.
