4.24. Encoding

Parameter Name Data Type Possible Values Default
ENCODING String   UTF-8
Table 4.25. ENCODING

  • With this parameter you can set the encoding for the command line/console of the machine on which your external command is executed

  • Possible values can be e.g. UTF-8 or 850.

  • You can use e.g chcp on Windows machines to find out the encoding of this very machine.

  • The encoding is especially important when your external command contains special characters, mutated vowels etc.
