Embedded Usage - Stardust Services

Embedded Usage

Embedded usage of Stardust allows to workflow-enable your application by leveraging functionality provided through the Stardust APIs.

Workflow functionality in Stardust is provided as a set of services in form of either plain Java Beans™ or Enterprise Java Beans™ (EJBs), thus allowing arbitrary Java™ applications to use the Stardust runtime functionality. In this way, Stardust can be used as an enabling technology, where it is solely the process engine that is integrated in the customer's proprietary embedding application. The embedding application has to communicate with a Stardust runtime environment.
