Stardust Schema

Momentarily Stardust supports two different formats for model files, the native Stardust format and XPDL, the XML-based Process Definition Language of the WfMC, which is the new standard Stardust file format. The Stardust Process Workbench is capable of loading and saving models in both formats. The models can be converted between these formats by saving them with extension .xpdl or .cwm for backward compatibility.
