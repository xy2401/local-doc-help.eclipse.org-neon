Internationalization of the Predefined Model

Internationalization of the Predefined Model can be achieved by using the predefined resource bundles.

Editing and adding Resource Bundles

Locale specific property files are supplied for the Predefined Model. They are called PredefinedModel_locale.properties and reside in the carnot-engine.jar file, folder ag/carnot.

Internationalization is provided for the following:

  • Model name and description
  • Case and case process details
  • Data like last activity performer details, process details etc.

In case a property file for your language already exist, edit the file with your locale extension. Otherwise copy the default or the English language file and rename it using your own locale. Edit the file and translate the entries accordingly.
