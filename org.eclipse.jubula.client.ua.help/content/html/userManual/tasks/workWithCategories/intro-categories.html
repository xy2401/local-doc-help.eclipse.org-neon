3.11.1. Introduction to categories

Categories let you group items together to be able to keep an overview of your data and find things more easily. You can use categories in the Test Case Browser, in the Test Suite Browser, in the Central Test Data Editor and in the Object Mapping Editor (see the section on object mapping for information on categories in the Object Mapping Editor (Section 3.19.2.3, “Working with categories in the Object Mapping Editor”)).
