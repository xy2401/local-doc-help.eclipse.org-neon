Support Case Example

This example contains the example models and the source code for the support case tutorial.

Click the following link to download the tutorial ZIP file containing the example sources in folder support-case:

  • all-tutorials.zip

The support case tutorial is a simple tutorial aiming and introducing you to all major aspects of modeling and programming with Stardust. It is not supposed to give you a detailed view of all Stardust functionalities, but will help you getting started with the software.

A detailed description of how to work with this example and a step by step description of how to create the support case model can be found in the Support Case Tutorial of the Stardust Tutorial Guide.
