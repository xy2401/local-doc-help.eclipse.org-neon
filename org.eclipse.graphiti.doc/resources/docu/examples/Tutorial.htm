Tutorial

The tutorial example provided along with the SDK download of Graphiti contains the final version of the Graphiti editor that is created in the Graphiti tutorial.

Tutorial Plugin

The example plugin org.eclipse.graphiti.examples.tutorial is available as executable but also as source in the SDK download and within our repository under the examples folder.

EMF Model

The tutorial uses the ECore model itself as domain model. Therefore there is no special model defined here.
