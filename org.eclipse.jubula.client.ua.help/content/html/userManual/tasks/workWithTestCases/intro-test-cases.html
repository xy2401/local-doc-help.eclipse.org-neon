3.12.1. Working with Test Cases

Test Cases are the ”building blocks” (keywords) for testing. They can:

  • contain other (referenced) Test Cases (Section 3.10.3, “Adding items to editors”)

  • contain Test Steps (Section 3.17.2, “Specifying Test Steps”)

  • be used in Test Suites (Section 3.10.3, “Adding items to editors”)

  • be used as Event Handlers (Section 3.26, “Dealing with errors in tests: Event Handlers”)
