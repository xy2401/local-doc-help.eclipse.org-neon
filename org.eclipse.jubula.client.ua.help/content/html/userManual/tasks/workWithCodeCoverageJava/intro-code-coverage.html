3.27.1. Working with code coverage with Java tests

The ITE is integrated with the JaCoCo code coverage tool for Java applications to let you run and analyze code coverage for your tests.

Code coverage is only available for AUTs started via an AUT configuration (Section 3.8.1, “Configuring AUTs to be started from the ITE”) (i.e. not using the autrun command), and which use Java 1.5 or higher.
