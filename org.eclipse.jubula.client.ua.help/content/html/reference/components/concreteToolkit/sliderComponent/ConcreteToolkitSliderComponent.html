3.2.7. Concrete toolkit slider component

A slider component is a component in which a data value can be chosen within a certain range.

The format of a slider component is a one-dimensional bar together with a knob which represents the chosen value.

Available Actions

  • Check Position (Slider Components)

  • Set Position (Slider Components)
