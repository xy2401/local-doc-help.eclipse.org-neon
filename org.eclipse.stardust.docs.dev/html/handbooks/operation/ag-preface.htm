Operation Guide

This guide describes the steps needed to configure the application server of choice for the deployment of the Stardust Process Engine, various possible configurations, how to monitor the runtime environment, to deploy models or manage users.

Audience

This guide is intended for administrators of the Stardust Process Engine as well as for modelers who want to deploy workflow models for production and using the Stardust Portal.

Contents

The remainder of this book contains the following issues:

  • Administration Clients describes the administration client provided by Stardust .
  • Command Line Tools describes the use of the SysConsole and Console command line tools for administration of the Stardust runtime.
  • Tuning describes the powerful tuning facilities provided by the Stardust Engine.
  • Security summarizes the configuration issues when integrating Stardust into an existing security infrastructure.
  • Logging describes logging possibilities in Stardust .
  • Audit Trail Model describes the usage of the schema of the model and audit trail database.
  • Stardust Properties describes the possible client- and server-side properties you can set in Stardust .
  • Internationalization describes the process of internationalizing Stardust components.
