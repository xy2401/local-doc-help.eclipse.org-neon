Delegating to a User in Another Target Department

This tutorial describes a simple use case to demonstrate the delegation from a user in department to a user in another target department.

Contents

To create the example Stardust model, start with the chapter Creating the Model and Organizational Structure. You can also import the example model DemoDelegationModel.xpdl provided by Stardust by extracting it from the scopedParticipants/dat folder residing in the following ZIP file:

all-tutorials.zip

In case you import the example model, you can skip the first chapter and continue with chapter Creating the Departments and Users in the Stardust Portal.
